# Core Track Notebooks

**Target Audience**: Students, educators, general learners interested in understanding the ETF flash crash

**Prerequisites**:
- Basic Python knowledge
- Understanding of stocks and ETFs (helpful but not required)
- No finance or math background needed

**Learning Path**:

## 1. Understanding the Crash (Start Here)
**File**: `01-understanding-the-crash.ipynb`

**What You'll Learn**:
- How ETFs are priced (NAV vs market price)
- What happened on August 24, 2015
- Why "stale" iNAV values caused problems
- How order book "air pockets" amplified the crash
- What LULD circuit breakers are and how they work

**Time**: 45-60 minutes
**Difficulty**: ⭐ Beginner

---

## 2. Real Market Data Analysis
**File**: `02-real-market-data-analysis.ipynb`

**What You'll Learn**:
- How to fetch real historical market data
- Analyzing the August 24, 2015 crash with actual prices
- Comparing to other market events (COVID crash, SVB crisis)
- Using Python tools for financial data analysis

**Time**: 30-45 minutes
**Difficulty**: ⭐⭐ Beginner-Intermediate

---
